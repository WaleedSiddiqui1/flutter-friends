Description:
Flutter Friends is a Java-based pet simulator that allows players to adopt and take care of their pets through 
various commands. The players can feed, play, give gifts, and take their pets to the vet while managing items 
and monitoring their pets' vitals, such as health, happiness, hunger, and sleep. The application uses object-oriented
programming principles to give players an interactive simulation experience.

Required Libraries and Tools:
- Java Development Kit(JDK) 19
- JUnit 5.10.0
- An IDE (Recommended: Intellij IDEA)
- Java Swing
- CSV File Support

Building the Software:
1) Install and set up JDK 19 
2) Clone project files into local directory
3) Open project in an IDE
4) Build the project using IDE's build tools

Running the Software:
1) Navigate the main class file (Main.java)
2) Run the program from the main class

User Guide:
- From the main menu click "New Game" and adopt a pet by selecting its type and name
- Interact with your pet by feeding it, playing with it, giving it gifts, or taking it to the vet
- Track your pet's stats using the fullness, sleep, happiness, and health bars
- To learn how to use each feature in detail, click the "Tutorial" button on the main menu page. This will guide you
though each command and help you learn all the game mechanics.
- You can also manage "Parental Controls" in the main menu to set limits on playtime and revive dead pets

Using the Jar File:
The program comes with an executable jar file that will run all functionalities of the main program.
However, in its current state the jar file does not correctly save information to file. So for now,
the version accessible via the jar file can only load the preset saves and create new saves. For the
full experience that includes the ability to save data to file, it is recommended to compile the
program as described above.

Additional notes:
When running via a compiler, the console will display various messages. These exist solely for the
purposes of debugging and should be ignored by users. The executable jar file does not display this
console. Occasional error messages may appear in the console, but these are minor errors that do not
at all affect the running of the program and can be ignored.
